# Finance Tracker (Qt 6 + SQLite)

Простое приложение для учета личных финансов. 
Написано на C++ с использованием фреймворка Qt 6 и базы данных SQLite.

## Основные возможности (Features)
- **Реляционная база данных**: использование SQLite для хранения транзакций, категорий и типов.
- **Динамическая фильтрация**: поиск по категориям и типам в реальном времени.
- **Интерактивные графики**: визуализация расходов с помощью QtCharts (Pie Chart).
- **Безопасность**: проверка баланса перед добавлением расходов.
- **Современный интерфейс**: использование Proxy-моделей для стабильной сортировки и удаления.

## Стек технологий
- C++17
- Qt 6 (Widgets, SQL, Charts)
- CMake
- SQLite

## Как запустить
1. Склонируйте репозиторий.
2. Откройте `CMakeLists.txt` в Qt Creator.
3. Соберите и запустите проект.
